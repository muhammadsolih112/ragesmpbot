# RageSMP — Telegram Mini App

RageSMP Minecraft serveri uchun maxsus ishlab chiqilgan Telegram Mini App ilovasi.

## 🚀 Xususiyatlari
- **Telegram Auto-login**: Foydalanuvchilar Telegram orqali avtomatik taniladi.
- **Dinamik Mavzu**: Telegram mavzusiga (Dark/Light) moslashish.
- **Donat Paketlari**: Server uchun donatlarni to'g'ridan-to'g'ri Mini App ichidan tanlash.
- **Admin Panel**: To'lovlar va foydalanuvchilarni boshqarish uchun qulay interfeys.
- **Haptic Feedback**: Ilova ichida professional vibratsiya effektlari.

## 🛠 Texnologiyalar
- **React 19**
- **Vite**
- **Tailwind CSS**
- **Telegram Web App SDK**

## 📦 O'rnatish
1. Bog'liqliklarni o'rnatish:
   ```bash
   npm install
   ```
2. Loyihani ishga tushirish:
   ```bash
   npm run dev
   ```

## 🌐 Deployment
Loyiha Vercel platformasi uchun optimallashtirilgan. GitHub repozitoriyasini Vercel'ga ulash kifoya.
